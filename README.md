# discord-court-bot
a very funny discord bot


.env
```
DISCORD_TOKEN=token
APPLICATION_ID=uwu
RUST_LOG=DEBUG
MONGO_URI=mongodb://localhost:27017
DB_NAME=court_bot
MONGO_INITDB_ROOT_USERNAME=root
MONGO_INITDB_ROOT_PASSWORD=uwu
DEV=
# SET_GLOBAL=
PRETTY=
```

run mongodb
```shell
docker compose up
```
